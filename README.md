# User Management App

## Description
A React application for managing user data with CRUD operations using Material-UI and Axios.

## Features
- Add new users
- Edit existing users
- Delete users
- Pagination
- Responsive design

## Prerequisites
- Node.js (v14 or later)
- npm

## Installation
1. Clone the repository
2. Run `npm install`
3. Run `npm start`

## Technologies Used
- React
- Material-UI
- Axios
- JSONPlaceholder API
